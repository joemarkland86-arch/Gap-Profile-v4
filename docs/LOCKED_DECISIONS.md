# Locked Decisions

Last updated: [7th March 2026]

Purpose: This file contains decisions that should not be reopened unless there is a strong reason. It prevents the project drifting or re-debating settled choices.

Product status: V1 development

- Four pillars:
  - Capability Belief
  - Risk Tolerance
  - Identity Fit
  - Capacity

- Four archetypes:
  - Capable Doubter
  - Risk Frozen
  - Identity Stretched
  - Empty Tank

- Core framing:
  Career development involves many elements, and even when people know the next
  step, taking action can still feel difficult. The Knowing–Doing Gap explains why.

- Tone:
  clear, grounded, structured, psychologically sharp, warm, direct

- Diagnostic:
  23 questions, radar chart, archetype result, email capture via Formspree
  Current version: V7 (forward-scored items, time anchors, updated band labels)

- Risk Frozen workbook:
  complete (24-page PDF, £9.99)

- Programme pricing:
  6-session programme at £597
  No 10-session option

- Results page structure:
  Archetype name → mechanism (short) → reflective question → locked interventions → CTA
  Micro-actions sit behind paywall (workbook or coaching), not on results page
